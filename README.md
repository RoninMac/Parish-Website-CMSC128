# Parish-Website-CMSC128
personal project/project requirement

How to run
1. make sure to have python installed and included in your path 
2. run to install django and create a virtual environment:
        pipenv install django
3. activate virtual environent: 
        pipenv shell
4. run the website 
        python manage.py runserver
